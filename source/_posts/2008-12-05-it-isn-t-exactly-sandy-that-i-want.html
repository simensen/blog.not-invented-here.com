Maybe she was close, though. I've been thinking about this quite a bit since I decided to move from I Want Sandy (will be shut down soon, so link may be broken) to Remember the Milk. Remember the Milk has a bunch of great features. In particular, rtm's Gmail Gadget seems to me to be far superior than their main web interface. The iPhone app ( pro feature ) is a huge bonus, too! Still, there was something very nice about being able to send a quick email to get things onto my calendar and todo list. Back when Twitter worked with IM, having a more or less realtime chat interface to Sandy via. Google Talk was amazing! I am going to be putting more thought into this, but I am starting to put together a list of things that would make an ideal app for myself. Maybe there is something out there already that does all of this? Not that I can see...

  • Closer to realtime sync with Google Calendar ( CalDAV? )
  • IM ( XMPP, Google chat, AIM, Y!M, Messenger )
  • SMS ( Txting, actual short code to use )
  • Gadgets (Gmail, Google Calendar, iGoogle, Google Desktop )
  • Email
