I have been having a lot of issues with web browsing lately. I am not yet sure if it is a problem with my internet connection (if so, I have the same problem with two separate providers, both at home and at the office), if it is a problem with my laptop (I hope not... not looking forward to reinstalling everything again...) or if it is a problem with Chrome.

Every time I try to switch back to FireFox from Chrome, I have an issue with the fact that I've grown so very accustomed to the "open link in new tab" being the first option in the right click menu and in FireFox, it is not. So, I did a quick search today for:

"open link in new tab" on top firefox

And what do you know? Courtesy of Chris Kite's blog, I found a link for a FireFox Menu Editor that let's me do exactly what I want. Swap the position of the "open link in" options.

Honestly, I'm hoping that FireFox has similar issues because I really have grown so used to Chrome. And I'm afraid of some of the other issues I've heard about this newer FireFox.
